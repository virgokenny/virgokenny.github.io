Error Code

The error number is used for checking the request command or parameters could be accepted by recorder service.

/*error_code*/
enum 
{
    XKD_OK = 0,
    XKD_FALSE = 1,
    XKD_IO_ERROR = 2,
    XKD_INVALID_HANDLE = 3,
    XKD_INVALID_PARAMS = 4,
    XKD_MISSING_PARAMS = 5
};

Recorder Status

Get the recorder status

get_status

The message is used for getting all metrics from recorder service.

  • Request parameters: None
  • Response parameters:
    • capture (object): Display the capture metrics.
    • playback (object): Display the playback metrics.
    • crops (object[]): Display the metrics of virtual channels.
      • index (string): The index of virtual channels.
      • uuid (string): The uuid of virtual channels.
      • metrics (object[]):
        • key (string): The key for the metric.
        • value (string): The value for the metric key.
    • metrics (object[]):
      • key (string): The key for the metric.
      • value (string): The value for the metric key.

set_device

The message is used for setting input(capture) device or output(playback) device in the recorder services.

  • Request parameters:
    • input_device (string): The selected index of capture device list.
    • output_device (string): The selected index of playback device list.
  • Response parameters:
    • error_code

Request example:

/*Args*/
[
    {
      "Type":"string",
      "Value":"--input_device 0 --output_device 20"
    }
]

Response example:

{
  "error_code":0
}
