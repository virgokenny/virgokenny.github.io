tmux

Init tmux

[xkd]$ tmux

Create window

ctl + b then c

Create Pane (Tile Split)

ctl + b then % # split the current pane vertically
ctl + b then " # split the current pane horizontally

Jump between Panes

ctl + b then press arrow key # move to other pane
ctl + b then o # jump to the next pane

Resize Pane

ctl + b then : resize-pane -D    # Resizes the current pane down
ctl + b then : resize-pane -U    # Resizes the current pane upward
ctl + b then : resize-pane -L    # Resizes the current pane left
ctl + b then : resize-pane -R    # Resizes the current pane right
ctl + b then : resize-pane -D 20 # Resizes the current pane down by 20 cells
ctl + b then : resize-pane -U 20 # Resizes the current pane upward by 20 cells
ctl + b then : resize-pane -L 20 # Resizes the current pane left by 20 cells
ctl + b then : resize-pane -R 20 # Resizes the current pane right by 20 cells

Close Pane

ctl + b then x # close the current pane

Show Pane ID

ctl + b then q # show the IDs of the panes

Zoom Pane

ctl + b then z # toggle pane zoom
